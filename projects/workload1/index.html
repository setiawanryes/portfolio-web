Workload Analyst

This project was carried out to analyze the workload of employees in the Accounting Sub-Department of KDK using the Task Management System (TMS) and Workload Analyst approach. The main objective was to evaluate workload balance, time efficiency, and task distribution effectiveness among employees.

The project was completed in approximately two months and involved eight accounting staff members. Data collection was conducted through direct observation of daily work activities, combined with the TMS approach to record, categorize, and monitor each assigned task.

Using the Workload Analyst method, the analysis compared actual workload distribution with the ideal capacity of each employee. The results were used to formulate recommendations for workflow improvement, including better task prioritization, balanced time allocation, and enhanced efficiency in accounting operations.

This project provides insights and practical strategies to help the Accounting Sub-Department of KDK achieve a more balanced, productive, and data-driven work environment.
